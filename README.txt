This project is part of the master course Ambient Intelligence at the University of Applied Sciences Kiel (Germany) in the summer semester 2013.

The students Nils Buerkner and Marco Neumann work at this project.

The project shall realize a bluetooth connection to a controller and shall visualize predefined data.
The controller is built into a Senseo Machine and can start making coffees, it can scan the cups in the machine,
which have an RFID-Chip, and send the data to the application which then visulaize the data.

The hole project description can be found as soon as possible under http://www.be-binary.de
